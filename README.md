# Time-To-Drop
Kasetsart University Students' Course Drop Recording Program

![](HomeScreen.png)
## Installation
   ```
   git clone https://github.com/ong22280/Time-To-Drop.git
   ```

## Run the program

```
python3 Drop.py
```

## Library used
- [tkinter](https://docs.python.org/3/library/tkinter.html)
